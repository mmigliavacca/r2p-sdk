# r2p-sdk

## Install

### Setup the SDK
    git clone https://github.com/r2p/r2p-sdk.git
    cd r2p-sdk
    git submodule init
    git submodule update

### Setup a demo workspace
    git clone https://github.com/r2p/workspace-demo.git workspace

## Usage

### Launch the Eclipse-based IDE
    ./r2p-ide.sh
